# Skillactive

Проект для поиска кружков

## **Стек технологий**

- HTML5;
- CSS3;
- Figma;
- Методология БЭМ;
- JavaScript;
- React.JS;
- Webpack

# **Установка и запуск проекта**

Клонировать репозиторий:

    git clone https://github.com/ronastushaaa/SkillActive
    
Установить зависимости:

    npm install


Запустить проект:

    npm start
